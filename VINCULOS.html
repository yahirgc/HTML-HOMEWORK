HIPERTEXTO

Sirven para ir hacia otro sitio web o para movernos dentro del mismo

Para movernos dentro del docuneto la etuiqueta es:

< a href = "www.example.com" target = "self" > anchor < / a>

anchor es la palabra que se ve para cliquear

El codigo para que se pueda identificar el area al que vamos debe tener esto

< a name = "www.example.com"> < / a>
